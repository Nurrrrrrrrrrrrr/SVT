Système nerveux

Les différentes parties du cerveau

Le cerveau est divisé en plusieurs parties distinctes :
On retrouve différents types de lobes :
-Le lobe frontal
-Le lobe pariétal
-Le lobe temporal
-Le lobe occipital
Ces lobes ont tous des particularités différentes, par exemple le lobe frontal est le siège de la parole, du langage et du raisonnement tandis que le lobe temporal est la zone où se situe la mémoire, le langage et l'émotivité.


La transmission du message nerveux

La transmission synaptique est le processus par lequel les neurones communiquent entre eux au sein du système nerveux, ce processus a lieu dans les synapses :
-La pré-synapse contient des vésicules synaptiques remplies de neuro-transmetteurs.
-Ce message voyage à travers la fente synaptique jusqu'aux éléments post-synaptique.
-La post-synapse étant équiper de récepteur spécifique reçoit le message nerveux.

Le neurone

Comme on peut le voir sur le schéma ci dessus, le neurone est une structure complexe a la fois receptrice d'un message nerveux et a la fois expeditrice de ce dernier, on arrive donc a decouper le neurone en 3 parties distinctes :

  1. Le corps cellulaire (soma) contrôle les activités cellulaires et génère des signaux électriques, deplus la dendrite joue le role de receptrice du message nerveux sous forme de signaux.

  2. L'axone est l'element du neurone permettant le transfert de l'information, le message va voyager le long de l'axone depuis le corps cellulaire vers les synapses.

  3. Le message est ainsi transmi depuis les terminaison nerveuses la ou se trouve les neuro-transmetteurs qui vont travers la fente synaptique.
